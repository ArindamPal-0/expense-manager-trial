# Expense Manager
### Full Stack django project

<br>

## Setup

### Dev Env Setup (Windows)
```Powershell
$ python -m venv .venv
$ .venv/Scripts/Activate.ps1
(.venv)$ pip install django
```

### To run the application
```Powershell
(.venv)$ python manage.py runserver
```
open the url on the browser to see the result.
http://127.0.0.1:8000
